# Ping-Pong-
Game of ping pong
Здесь вы можете поиграть с другом в пинг понг. Есть два игрока у одного управляется на кнопки A D у другого на Right Left (стрелки)также присутствует счетчик игра идет до 5 очков кто набрал тот выиграл

English:
Ping Pong Game: Play ping pong with a friend. There are two players, one using the A and D keys, the other using the Right and Left arrow keys. There's also a counter, and the game goes up to 5 points, and the player who scores the most wins.
